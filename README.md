# 👟 SyntaxWear - Tênis e Sneakers Online

Bem-vindo ao repositório da **SyntaxWear**! Este é um projeto de e-commerce focado na venda de calçados modernos e exclusivos, desenvolvido como parte de um desafio de aprendizado em desenvolvimento front-end.

![SyntaxWear Preview](./images/logo/logo.svg)

---

## 📝 Sobre o Projeto

O SyntaxWear é uma landing page moderna para uma loja virtual de sneakers. O objetivo principal do projeto foi praticar a estruturação semântica de HTML5 e a estilização modular com CSS3, garantindo um visual profissional e uma interface amigável para o usuário.

## 🚀 Funcionalidades

- **Design Moderno**: Interface inspirada nos grandes e-commerces de calçados.
- **Navegação Responsiva**: Menu hambúrguer preparado para dispositivos móveis.
- **Seção Hero**: Destaque principal com chamadas para ação (CTAs).
- **Categorização Inteligente**: Áreas dedicadas para estilos Casual, Esporte, Moderno e Futurista.
- **Grade de Produtos**: Exibição organizada dos modelos em destaque (Krypton One).
- **Newsletter e Rodapé**: Área para inscrição de e-mail e links sociais organizados por categorias.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica de todo o conteúdo.
- **CSS3**: Estilização modularizada, utilizando:
  - **Variáveis CSS**: Centralização de fontes e cores.
  - **Flexbox & Grid**: Para layouts complexos e alinhamento de componentes.
  - **Google Fonts**: Fonte 'Ubuntu' integrada para uma tipografia limpa.

## 📁 Estrutura de Arquivos

O projeto segue uma organização modular para facilitar a manutenção:

```text
ecommerce-syntaxwear/
├── css/
│   ├── components/       # Estilos específicos de cada seção (header, footer, etc.)
│   ├── base.css          # Estilos globais e padrões
│   ├── reset.css         # Reset de estilos do navegador
│   └── variables.css     # Definição de cores e fontes (Custom Properties)
├── images/
│   ├── banners/          # Imagens de fundo da seção Hero
│   ├── icons/            # Ícones em SVG (sacola, usuário, redes sociais)
│   ├── logo/             # Logo oficial da SyntaxWear
│   └── products/         # Fotos dos modelos de tênis
└── index.html            # Estrutura principal da página
```

## 💻 Como Rodar o Projeto

Como este é um projeto estático (HTML e CSS), você não precisa de instaladores.

1. Faça o download ou clone este repositório.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox, etc.).
